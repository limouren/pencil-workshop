# Chimapp workshop content pack

Curated default content for the **AI-Assisted Web Design with Pencil** workshop.

Use this pack when participants do not bring their own website content. It is intentionally small: copy for a multi-page website, concise service copy, three blog cards, contact copy, and a small set of visual assets.

## Recommended workshop outcome

Build and deploy a responsive multi-page website for **Chimapp**, a software development and testing studio led by two cats.

## Files

- `01-workshop-brief.md` — brand, audience, page structure, and asset suggestions.
- `02-homepage-copy.md` — copy for the homepage.
- `03-services-copy.md` — concise service card copy and optional detail blurbs.
- `04-pricing-copy.md` — pricing cards.
- `05-blog-preview-copy.md` — three curated blog cards.
- `06-contact-copy.md` — contact CTA, form labels, social links, and footer copy.
- `data/blog-posts.csv` — the same three blog cards in CSV format.
- `assets/public/images/` — logo, homepage images, and social icons.
- `assets/public/blogs/hero-images/` — curated blog preview images.

## Suggested build scope

Build the workshop site as a small multi-page website:

1. **Home** (`/`) — header, hero, services overview, about, pricing preview, latest articles, contact CTA, footer
2. **Services** (`/services`) — detailed service cards
3. **Pricing** (`/pricing`) — pricing cards
4. **Blog** (`/blog`) — blog post listing using the curated cards
5. **Contact** (`/contact`) — contact form, social links, and contact info

Keep the blog simple: use the provided cards as a listing page; do not build a full CMS during the workshop.
