# Chimapp workshop content pack

Curated default content for the **AI-Assisted Web Design with Pencil** workshop.

Use this pack when participants do not bring their own website content. It is intentionally small: one landing page, concise service copy, three blog cards, contact copy, and a small set of visual assets.

## Recommended workshop outcome

Build and deploy a responsive one-page website for **Chimapp**, a software development and testing studio led by two cats.

## Files

- `01-workshop-brief.md` — brand, audience, page structure, and asset suggestions.
- `02-homepage-copy.md` — copy for the landing page.
- `03-services-copy.md` — concise service card copy and optional detail blurbs.
- `04-pricing-copy.md` — pricing cards.
- `05-blog-preview-copy.md` — three curated blog cards.
- `06-contact-copy.md` — contact CTA, form labels, social links, and footer copy.
- `data/blog-posts.csv` — the same three blog cards in CSV format.
- `assets/public/images/` — logo, homepage images, and social icons.
- `assets/public/blogs/hero-images/` — curated blog preview images.

## Suggested build scope

Keep the workshop site to one page:

1. Header
2. Hero
3. Services
4. About
5. Pricing
6. Latest articles
7. Contact CTA
8. Footer

Do not build a full blog system during the workshop. Use the blog content only as preview cards unless participants have extra time.
