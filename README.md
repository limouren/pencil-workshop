# AI-Assisted Web Design with Pencil — Workshop Materials

Materials for the **AI-Assisted Web Design with Pencil** workshop at [Taiwan Digital Fest (TDF)](https://fest.dna.org.tw/).

Event page: <https://luma.com/btnn6bnf>

## Goal

Design a website in [Pencil](https://pencil.dev/) and deploy it to GitHub Pages.

## Before you arrive

### Required

- [ ] A laptop with **macOS** or **Linux** (Windows works but with limited helper support)
- [ ] An active **Claude Code** or **Codex** subscription
- [ ] A **GitHub** account (we can help you create one at the workshop)

### Optional

- [ ] Your own website content (text, images, copy). Otherwise, use the default content pack below.

## What's in this repo

### `website-content/` — Default content pack

A ready-to-use content set for **Chimapp**, a fictional software studio led by two space cats.

| File | Contents |
|------|----------|
| `01-workshop-brief.md` | Brand, audience, asset list |
| `02-homepage-copy.md` | Homepage copy — hero, nav, about, footer |
| `03-services-copy.md` | Six service cards with descriptions |
| `04-pricing-copy.md` | Four pricing tiers |
| `05-blog-preview-copy.md` | Three blog preview cards |
| `06-contact-copy.md` | Contact form, social links, footer copy |
| `data/blog-posts.csv` | Blog cards in CSV format |
| `assets/` | Logo, team photos, blog hero images, social icons |

### Suggested page structure

1. **Home** (`/`) — Hero, services overview, about, pricing preview, blog cards, contact CTA
2. **Services** (`/services`) — Detailed service cards
3. **Pricing** (`/pricing`) — Pricing tiers
4. **Blog** (`/blog`) — Blog post listing
5. **Contact** (`/contact`) — Contact form and social links

We'll focus on a **single-page landing site** first. Multi-page comes after if time permits.

## Workshop outline

1. **Generate** — Create a page design from a text prompt in Pencil
2. **Refine** — Edit and iterate on the design
3. **Deploy** — Ship to GitHub Pages
